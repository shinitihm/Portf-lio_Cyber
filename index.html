Este projeto orienta a criação de um portfólio coletivo no GitHub. O exercício envolve a configuração do repositório, onde olíder cria o projeto com um arquivo README e convida os colegas como colaboradores e cada colega cria seu próprio portfólio. Por fim, o líder do repositório faz deploy do GitHub pages e cada colega faz um fork do repositório.
